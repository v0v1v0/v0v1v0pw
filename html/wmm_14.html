.DeriveVersionInfo R Documentation

Derive WMM version based on given time

Description

Derive WMM version based on given time

Usage

.DeriveVersionInfo(t)

Arguments

t

Annualized date time. E.g., 2015-02-01 = (2015 + 32/365) = 2015.088

Value

List of reference year and compatible WMM versions inferred from time.
