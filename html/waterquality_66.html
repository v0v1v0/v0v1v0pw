TurbLath91RedOverBlue R Documentation

TurbLath91RedOverBlue algorithm

Description

Applies the TurbLath91RedOverBlue algorithm

Usage

TurbLath91RedOverBlue(w658, w458)

Arguments

w658

numeric. Value at wavelength of 658 nm
w458

numeric. Value at wavelength of 458 nm

Value

SpatRaster or numeric

References
