eurosoccer R Documentation

European Soccer Leagues

Description

Contains various team stats from five European soccer leagues (2008/09 season).

Usage

eurosoccer

Format

A data frame with 11 variables and 96 teams:

League

Country

Team

Team

Games

Number of games

Won

Games won

Tied

Games tied

Lost

Games lost

GoalsScored

Goals scored

GoalsConceded

Goals conceded

GoalDifference

Goal difference

Points

Final amount of points

GoalsGame

Goal scored per game

Examples

head(eurosoccer)
