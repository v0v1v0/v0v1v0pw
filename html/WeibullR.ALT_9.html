options.alt R Documentation

Options list for alt Objects

Description

This function handles the various plotting options for alt objects.

Usage

options.alt(...)

Arguments

...

Options for plotting alt objects using the WeibullR package.

Details

As a function, options.alt, borrows its internal structure from the par function of package graphics.

Unlike its corresponding function in WeibullR options.wblr this function is intended to only be called within plot.alt.

Value

Executing options.alt without arguments returns a named list containing the default options for plotting a alt object.
Should arguments be supplied, these are returned in a named list.
