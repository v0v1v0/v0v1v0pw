wgport R Documentation

Get webglobe's port

Description

Determine which port a webglobe is running on

Usage

wgport(wg)

Arguments

wg

Webglobe whose port should be determined

Value

A number representing the webglobe's port

Examples

## Not run: 
library(webglobe)
wg<-webglobe(immediate=TRUE)
wgport(webglobe)

## End(Not run)
