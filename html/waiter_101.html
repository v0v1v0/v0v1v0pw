steward R Documentation

Steward

Description

A colorful steward to work with the waiter.

Usage

useSteward(
  colors = c("#ee7752", "#e73c7e", "#23a6d5", "#23d5ab"),
  speed = 30,
  angle = -45
)

use_steward(
  colors = c("#ee7752", "#e73c7e", "#23a6d5", "#23d5ab"),
  speed = 30,
  angle = -45
)

Arguments

colors

Color palette forming gradient.
speed

Seconds it takes to loop over colors.
angle

Degrees at which colors slide.
