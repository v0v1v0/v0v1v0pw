is_outlier R Documentation

Flag Outliers

Description

Returns a logical vector that indicates which observations were declared outlier by the method.

Usage

is_outlier(object, ...)
## S3 method for class 'wbaconlm'
is_outlier(object, ...)
## S3 method for class 'wbaconmv'
is_outlier(object, ...)

Arguments

object

object of class wbaconmv or wbaconlm.
...

additional arguments passed to the method.

Value

A logical vector.

See Also

wBACON_reg and wBACON

Examples

data(swiss)
m <- wBACON(swiss)
is_outlier(m)
