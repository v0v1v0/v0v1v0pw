HardThresh R Documentation

Hard Threshold

Description

Apply hard threshold.

Usage

HardThresh(y, t)

Arguments

y

vector
t

threshold

Value

vector $y$ thresholded: entries below $t$ are replaced by zeros.

Author(s)

Marc Raimondo

References

Raimondo, M. and Stewart, M. (2007), "The WaveD Transform in R", Journal of Statistical Software.

See Also

WaveD

Examples

HardThresh(1:5,2)
