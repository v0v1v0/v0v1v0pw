phenotype1 R Documentation

Phenotype of the diabetes.geno Data

Description

A binary variable indicate the status of 115 individuals (1 = affacted, 0 = unaffacted).

Usage

data(phenotype1)

Format

The format is: 115 rows and 1 column.
