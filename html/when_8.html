generate_table R Documentation

Generate table

Description

Once all the characteristics of the dimension have been defined, we can generate its table according to them using this function.

Usage

generate_table(td)

## S3 method for class 'when'
generate_table(td)

Arguments

td

A when object.

Value

A when object.

td <- when() |>
