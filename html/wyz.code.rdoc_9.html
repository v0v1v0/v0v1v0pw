generateEnc R Documentation

Generate Enc

Description

Generate special markup for encoding text in R documentation

Usage

generateEnc(entries_l)

Arguments

entries_l

An unconstrained named list of listwith text and ascii names, holding respectively international text and ASCII equivalent.

Value

A list of strings

Author(s)

Fabien Gelineau <neonira@gmail.com>

Maintainer: Fabien Gelineau <neonira@gmail.com>

Examples

generateEnc(list(list(text = 'Français', ascii = 'Francais')))
# [[1]]
# [1] "\enc{Français}{Francais}"

generateEnc(list(list(text = 'é', ascii = 'e'), list(text = 'è', ascii = 'e')))
# [[1]]
# [1] "\enc{é}{e}"

# [[2]]
# [1] "\enc{è}{e}"
