set.as R Documentation

WRSS objects connector

Description

this function connects a base object as a either of: 'downstream', 'supplier', 'leakageObject', 'seepageObject', or 'divertObject' to a target object, which are both instantiated by WRSS constructors.

Usage

set.as(base,target,type='downstream')

Arguments

base

An object; from either of classes of createAquifer , createRiver, createReservoir, createJunction, createDiversion, or createDemandSite
target

An object; from either of classes of createAquifer , createRiver, createReservoir, createJunction, createDiversion, or createDemandSite
type

the type of base object to be set as to the target object: 'downstream', 'supplier', 'leakageObject', 'seepageObject', or 'divertObject'

Value

an object from class of target object.
