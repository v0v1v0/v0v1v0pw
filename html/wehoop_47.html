request_with_proxy R Documentation

Retry http request with proxy

Description

This is a thin wrapper on httr::RETRY

Usage

request_with_proxy(
  url,
  params = list(),
  origin = "https://stats.wnba.com",
  referer = "https://www.wnba.com/",
  ...
)

Arguments

url

Request url
params

list of params
origin

Origin url
referer

Referer url
...

passed to httr::RETRY
