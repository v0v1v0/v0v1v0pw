WColorBarH R Documentation

WColorBarH

Description

a horizontal color bar

Usage

WColorBarH(
  data,
  ...,
  label = NULL,
  label.side = "r",
  label.fontsize = 12,
  label.pad = 0.005,
  label.space = 0.05,
  label.use.data = FALSE
)

Arguments

data

numeric vector
...

additional options to WHeatmap
label

colorbar label
label.side

l (for left) or r (for right)
label.fontsize

label font size
label.pad

label padding
label.space

when label.use.data, the space between labels
label.use.data

use data to show legend in situ

Value

an object of class WColorBarH

Examples

WColorBarH(matrix(1:50))
