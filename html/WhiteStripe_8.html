s.hist R Documentation

Smoothed histogram of image

Description

Smoothed histogram of image

Usage

s.hist

Format

A GAM from mgcv for x and y from histograms

Examples

## Not run:  
data(t2.voi.hist)
y = t2.voi.hist$counts
x = t2.voi.hist$mids
x = x[!is.na(y)];
y = y[!is.na(y)]
# 70 used for speed of example
s.hist = smooth_hist(x, y, k=70)

## End(Not run)
