pdf_kable R Documentation

Enhanced knitr::kable with latex

Description

pdf_kable formats tibbles/df's for markdown

Usage

pdf_kable(
  .input,
  width1 = 6,
  twidth = 14,
  tposition = "left",
  innercaption = NULL,
  caption = "",
  foot = NULL,
  escape = TRUE
)

Arguments

.input

table to print
width1

Width of 1st column, default 6.
twidth

Default 14
tposition

Default left
innercaption

subheader
caption

header
foot

footnote
escape

see knitr::kable

Value

A character vector of the table source code.
