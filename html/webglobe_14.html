wgtitle R Documentation

Title of webglobe browser window

Description

Changes the tab/window title of the webglobe's browser view

Usage

wgtitle(title)

Arguments

title

The title to use

Value

A webglobe command

Examples

## Not run: 
library(webglobe)
wg<-webglobe(immediate=TRUE)
wg+wgtitle("I am the new title!")

## End(Not run)
