DEFF R Documentation

Calculate DEFF

Description

Calculates design effect (DEFF)

Usage

DEFF(x)

Arguments

x

= weights vector (name of weights column)

Value

Design effect (DEFF)

References

Design effect (DEFF) due to weighting => n * (sum(x^2) / sum(x)^2)

Examples

DEFF(testweights$weights_column)
