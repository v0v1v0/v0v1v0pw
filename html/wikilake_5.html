lake_wiki

Description

lake_wiki

Usage

lake_wiki(lake_name, map = FALSE, clean = TRUE, ...)

Arguments

lake_name

character
map

logical produce map of lake location?
clean

logical enforce standardized units following wikilake::unit_key_()?
...

arguments passed to maps::map
