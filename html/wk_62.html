Description

Grid information

Usage

grd_summary(grid)

Arguments

grid

A grd_xy(), grd_rct(), or other object implementing ⁠grd_*()⁠ methods.

Value

  • grd_summary() returns a list() with components xmin, ymin, xmax, ymax, nx, ny, dx, dy, width, and height.

Examples

grd_summary(grd(nx = 3, ny = 2))
