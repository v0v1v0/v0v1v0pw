wbt_launch_runner R Documentation

Launch 'WhiteboxTools Runner' GUI

Description

wbt_launch_runner(): Launch the 'WhiteboxTools Runner' GUI at wbt_runner_path()

Usage

wbt_launch_runner(clear_app_state = FALSE)

Arguments

clear_app_state

Clear application state memory? Default: FALSE

Details

Opens the 'WhiteboxTools Runner' GUI included with WhiteboxTools Open Core v2.3.0 or higher.
