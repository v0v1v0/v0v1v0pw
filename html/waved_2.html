dyad R Documentation

Dyadic band

Description

Returns a vector of integers 2^j+1,\ldots,2^{j+1}.

Usage

dyad(j)

Arguments

j

Resolution Level

Value

Returns a vector of integers 2^j+1,...,2^{j+1}.

Author(s)

Marc Raimondo

See Also

FWaveD

Examples

library(waved)
data=waved.example(TRUE,FALSE)
lidar.w=FWaveD(data$lidar.blur,data$g)
lidar.w[dyad(7)]
