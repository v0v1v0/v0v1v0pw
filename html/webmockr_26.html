remove_request_stub R Documentation

Remove a request stub

Description

Remove a request stub

Usage

remove_request_stub(stub)

Arguments

stub

a request stub, of class StubbedRequest

Value

logical, TRUE if removed, FALSE if not removed

See Also

Other stub-registry: StubRegistry, stub_registry(), stub_registry_clear()

Examples

(x <- stub_request("get", "https://httpbin.org/get"))
stub_registry()
remove_request_stub(x)
stub_registry()
