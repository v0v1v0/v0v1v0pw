grepv R Documentation

Return a vector of matches.

Description

Return a vector of matches.

Usage

grepv(
  pattern,
  x,
  ...,
  ignore.case = FALSE,
  perl = FALSE,
  fixed = FALSE,
  useBytes = FALSE,
  invert = FALSE
)

Arguments

pattern

character scalar, pattern to match, passed to grep.
x

character vector to match to, passed to grep.
...

not used, forced later arguments to bind by name.
ignore.case

logical, passed to grep.
perl

logical, passed to grep.
fixed

logical, passed to grep.
useBytes

logical, passed grep.
invert

passed to grep.

Value

vector of matching values.

See Also

grep, grepdf

Examples


grepv("x$", c("sox", "xor"))
