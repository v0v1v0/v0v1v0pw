player_or_team

player_scope

Player Scope - All Players, Rookies
season

Season - format 2020-21
season_type

Season Type - Regular Season, Playoffs
stat_type

Stat Type - Traditional, Advanced, Tracking
...

Additional arguments passed to an underlying function like httr.
